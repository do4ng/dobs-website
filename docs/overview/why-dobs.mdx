Dobs is a lightweight framework for building backend applications.
It aims to stay minimal while still providing powerful capabilities.

## Key Features

- **Fully TypeScript support**: Uses [rolldown](https://rolldown.rs) to support TypeScript.
- **Automatic server reload**: No need to restart the server when making changes.
- **File-based routing**: Operates based on file names.
