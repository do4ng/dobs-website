The logger provides a simple way to record basic data such as timestamps, request URLs, status codes, and elapsed time.

## Installation

1. Install via your package manager.

```bash
yarn add -D @dobsjs/logger
```

2. Update your [dobs.config.ts](/docs/config-file).

```ts
import { logger } from "@dobsjs/logger";
import { defineConfig } from "dobs";

export default defineConfig({
  plugins: [logger()], // [!code ++]
});
```

## Example Output

```bash
2025-12-01T12:12:17.149Z    GET /                        200  18.59ms
2025-12-01T12:12:17.286Z    GET /favicon.ico             200  12.38ms
```
